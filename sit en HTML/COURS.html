introduction au langage html

Commencer avec le HTML

Dans son cœur, HTML est un langage vraiment simple, comp osé d’éléments appliquables à des fragments de texte dans un document pour leur donner un sens différent (est-ce un pa ragraphe ? est-ce une liste à puces ? est-ce une partie de tableau ?), pour structurer un document en sections logiques (a‑t‑il un en-tête ? est-il sur t rois colonnes ? a-t-il un menu de navigation ?) et pour intégrer du contenu comme images ou vidéos dans une page. Ce module est une introduction aux deux premiers concepts ; il présente les notions fondamentales et la syntaxe à connaître pour comprendre le HTML.

Au fond, le HTML est un langage assez simple composé d' éléments , qui peut être appliqué à des morceaux de texte pour leur donner une signification différente dans un document (S'agit-il d'un paragraphe? Est-ce une liste à puces? Est-ce que ça fait partie d'un tableau? ), structurez un document en sections logiques (a-t-il un en-tête? Trois colonnes de contenu? Un menu de navigation?), et intégrez du contenu tel que des images et des vidéos dans une page. Ce module présentera les deux premiers d'entre eux et présentera les concepts fondamentaux et la syntaxe que vous devez connaître pour comprendre le HTML.

Liens HTML

Remarque: un lien ne doit pas nécessairement être du texte. Un lien peut être une image ou tout autre élément HTML!

Les liens se trouvent dans presque toutes les pages Web. Les liens permettent aux utilisateurs de cliquer de page en page. Les liens HTML sont des hyperliens. Vous pouvez cliquer sur un lien et passer à un autre document. Lorsque vous déplacez la souris sur un lien, la flèche de la souris se transforme en une petite main.

L'attribut cible: Par défaut, la page liée sera affichée dans la fenêtre actuelle du navigateur. Pour changer cela, vous devez spécifier une autre cible pour le lien.
L' targetattribut spécifie où ouvrir le document lié.
L' targetattribut peut avoir l'une des valeurs suivantes:
_self- Défaut. Ouvre le document dans la même fenêtre / onglet où il a été cliqué
_blank - Ouvre le document dans une nouvelle fenêtre ou un nouvel onglet
_parent - Ouvre le document dans le cadre parent
_top - Ouvre le document dans le corps entier de la fenêtre.

Liens HTML - Créer des signets

Les liens HTML peuvent être utilisés pour créer des signets, afin que les lecteurs puissent accéder à des parties spécifiques d'une page Web. Les signets peuvent être utiles si une page Web est très longue. Pour créer un signet - créez d'abord le signet, puis ajoutez un lien vers celui-ci. Lorsque vous cliquez sur le lien, la page défile vers le bas ou vers le haut jusqu'à l'emplacement avec le signet. Créer des signets

Images HTML

Les images peuvent améliorer la conception et l'apparence d'une page Web.

La "img"balise HTML est utilisée pour incorporer une image dans une page Web.
Les images ne sont pas techniquement insérées dans une page Web mais les images sont liées à des pages Web.
La "img" balise crée un espace de stockage pour l'image référencée.
La "img"balise est vide, elle contient uniquement des attributs et n'a pas de balise de fermeture.
La "img "balise a deux attributs obligatoires:

src - Spécifie le chemin vers l'image
alt - Spécifie un autre texte pour l'image

L'attribut src

Remarque: lorsqu'une page Web se charge; c'est le navigateur, à ce moment-là, qui récupère l'image d'un serveur Web et l'insère dans la page. Par conséquent, assurez-vous que l'image reste au même endroit par rapport à la page Web, sinon vos visiteurs recevront une icône de lien cassé. L'icône de lien rompu et le alttexte s'affichent si le navigateur ne trouve pas l'image.

Tableaux HTML

Les tableaux HTML permettent aux développeurs Web d'organiser les données en lignes et en colonnes.

Cet attribut ne doit pas être utilisé car il a été déprécié : l'élément "table" devrait être stylisé en utilisant CSS. Pour obtenir un effet similaire à celui réalisé par l'attribut align, les propriétés CSS text-align et vertical-align devraient être utilisées. Avant Firefox 4, Firefox supportait également, en mode quirks uniquement, les valeurs middle, absmiddle, et abscenter comme synonymes de center.

Utilisez l' "tr"élément HTML pour définir une ligne de tableau
Utilisez l' "td"élément HTML pour définir les données d'un tableau

Utilisez l' "caption"élément HTML pour définir une légende de tableau

Utilisez la "border-collapse"propriété CSS pour réduire les bordures de cellule
Utilisez la "padding"propriété CSS pour ajouter un remplissage aux cellules

Utilisez la "border-spacing"propriété CSS pour définir l'espacement entre les cellules

Audio HTML

L' "audio"élément HTML est utilisé pour lire un fichier audio sur une page Web.

- Méthodes, propriétés et événements :Le DOM HTML définit des méthodes, des propriétés et des événements pour l' "audio"élément. Cela vous permet de charger, lire et mettre en pause des audios, ainsi que de définir la durée et le volume.
Il existe également des événements DOM qui peuvent vous avertir lorsqu'un audio commence à jouer, est mis en pause, etc.
Pour une référence DOM complète, consultez notre référence DOM HTML Audio / Vidéo . Balises audio HTML:
audio:Defines sound content
source:Defines multiple media resources for media elements, such as "video" and "audio"

Audio HTML - Comment ça marche

L' "controls" attribut ajoute des commandes audio, telles que la lecture, la pause et le volume.
L' "source" élément vous permet de spécifier des fichiers audio alternatifs parmi lesquels le navigateur peut choisir.
Le navigateur utilisera le premier format reconnu.
Le texte entre les balises "audio"et "/audio"ne sera affiché que dans les navigateurs qui ne prennent pas en charge l' "audio"élément.

Vidéo HTML

L' "video"élément HTML est utilisé pour afficher une vidéo sur une page Web.

L' controlsattribut ajoute des commandes vidéo, telles que la lecture, la pause et le volume. Il est une bonne idée d'inclure toujours widthet heightattributs. Si la hauteur et la largeur ne sont pas définies, la page peut scintiller pendant le chargement de la vidéo. L' "source"élément vous permet de spécifier des fichiers vidéo alternatifs parmi lesquels le navigateur peut choisir. Le navigateur utilisera le premier format reconnu. Le texte entre les balises "video"et "/video"ne sera affiché que dans les navigateurs qui ne prennent pas en charge l' "video" élément.

Balises vidéo HTML

"vidéo" Définit une vidéo ou un film
"source" Définit plusieurs ressources multimédias pour les éléments multimédias, tels que "vidéo" et "audio"
"track" Définit les pistes de texte dans les lecteurs multimédias
